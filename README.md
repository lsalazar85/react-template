# React Template
React template for Nucba students

## Run
```
    npm install
    npm run start
```

```
    yarn install
    yarn run start
```
Starts on `http://localhost:3000`

## Build
```
    npm run build
```

## Development technologies and external packages
* React 
* Jest & React testing library
* React Router
* Styled Components
* Node sass
* Axios
* Redux
